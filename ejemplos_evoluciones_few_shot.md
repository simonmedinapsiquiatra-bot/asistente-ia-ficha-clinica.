# Ejemplos de Evoluciones Psiquiátricas (Few-Shot Prompting)

El siguiente documento contiene ejemplos reales de evoluciones clínicas psiquiátricas redactadas con un estilo particular. El propósito de este archivo es servir como "Few-Shot Prompting" para que el modelo de IA aprenda el tono, la estructura, la terminología médica, las abreviaturas y la forma de redacción.
Todos los nombres de pacientes, RUTs, profesionales u otros datos identificables han sido reemplazados por etiquetas como `[NOMBRE PACIENTE]`, `[RUT PACIENTE]`, `[NOMBRE DOCTOR]`, etc., para preservar el anonimato.
